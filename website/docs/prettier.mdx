You can create a `.prettierrc` file in the root directory of your project and paste the following configuration:

```.prettierrc.json
{
  "$schema": "http://json.schemastore.org/prettierrc",
  "printWidth": 80,
  "semi": true,
  "singleQuote": true
}
```

Adjust the configuration options as needed based on your project's requirements.

### Configure

Reference the configuration in your `package.json` file:

```json
{
  "prettier": "@prettier/prettier-config"
}
```

### Run

Add a script in your `package.json` file to run Prettier:

```json
{
  "scripts": {
    "lint": "prettier -c --cache ."
  }
}
```
