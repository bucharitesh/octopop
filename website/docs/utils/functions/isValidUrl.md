# Function: isValidUrl()

> **isValidUrl**(`url`): `boolean`

Checks if a given string is a valid URL.

## Parameters

• **url**: `string`

The URL to validate.

## Returns

`boolean`

true if the URL is valid, false otherwise.

## Example

```ts
isValidUrl('https://example.com'); // true
isValidUrl('not-a-url'); // false
```
